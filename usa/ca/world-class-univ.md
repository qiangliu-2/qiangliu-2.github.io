# 加州之世界一流高等教育

李遠哲、钱永健、朱棣文有什么共同之处？他们获得诺贝尔奖的时候都在**美国加州**的大学工作。

作者比较熟悉的诺奖得主，有测量电子质量的Millikan、发明量子路径积分的Feynman及化学大师Pauling，都出自加州理工。

Pauling绝对是个传奇。他本人两次独立获得诺奖，还有众多学术传人获奖。他的学生、诺奖得主William Lipscomb是作者博士导师诺奖得主的导师。他的另一个学生、诺奖得主Martin Karplus是作者博士后导师的导师。李遠哲也可以说是Pauling的[学术传人](#Notes)。

加州除了斯坦福大学、加州理工及伯克利这些世界顶级大学之外，还有其它六所大学的[教员](#Notes)获得科学类（物理、化学、生理医学）诺奖，包括加州大学圣地亚哥分校和南加州大学。

加州的大学共80次获得科学类诺奖。如果把加州当作一个国家，**加州获得科学类诺奖的人数仅次于美国，排世界第二**。从科学类诺奖这个指标来看，加州无疑具有世界一流的高等教育。
